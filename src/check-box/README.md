# Checkbox component
    - Create a check box component

## Checkbox:

### Depedencies: none

### Props:
 - style?: StyleProp<ViewStyle>;
 - outlineStyle?: StyleProp<ViewStyle>;
 - fillStyle?: StyleProp<ViewStyle>;
 - disable?: boolean;
 - initialValue?: boolean;
 - value?: boolean;
 - text?: string;
 - onToggle?: (newValue: boolean) => void;
